## Author
Tasnim Rahman

## Description
This program creates a yellow background window with my full name "Tasnim Rahman" as the window title.Inside the window, a cyan-colored star is drawn using only triangles.
When the user presses the first letter of my name ('T') from the keyboard, the window closes automatically.

## Language & Tools
1.Language: C++
2.Graphics Library: OpenGL (GLUT)
3.Editor: VS Code
4.Operating System: Windows

## How to Compile and Run
1. Open the terminal in the project folder
2. Compile the program:
   main.cpp
3. Run the program:
   main.cpp

## Key Functionality
- Window background color → Yellow
-Star color → Cyan
-Star is built only with GL_TRIANGLES
-Window title → Tasnim Rahman
-Press key 'T' to close the window
